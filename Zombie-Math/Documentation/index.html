How to Install and Configure Backend

1. Login to your Cpanel and go to MySQL Database

2. Create a new Database with any name.

3. Create a new MySQL user.

4. Add that MySQL user to the newly created database.

5. Now go back and go to phpmyadmin and select the database that you just created from the left side.

6. From the navigation bar, select import and upload any one of two sql files from "Backend Source Code". The "zombiemath_empty.sql" file is an empty database file, you need add new questions to it to make the game work. The "zombiemath_with_data.sql" file is the database file filled with data that you saw in the demo, you can add new questions to it.

7. Once uploaded, you will these two tables in your database. Now go back to the cpanel

8. Go to file manager and upload the backend and the new configured frontend game files. Once you have both the files, go to backend/core and right click on config.php and select edit.

9.The editor will open in another window. Now edit the values that I have blacked out in the image below with your database details, you can find these details in the MySQL database.

10.Once done, save the changes and close all the cpanel tabs.

11.You have successfully installed the game and configured the backend!

12. Now go to yourwebsite.com/zombiemath to play the game and you can add new questions by going to yourwebsite.com/zombiemath/backend and login there.

How to add Admob

Steps to add Admob

This Game is Compatible with AdmobAds.

1.Create or select a Google Account.

2.Go to www.admob.com and sign up an account.

3.Once logged in, Click on 'Monetize New App' button.4.After Clicking 'Monetize New App' button,you will see a window like this :

4.After Clicking 'Monetize New App' button,you will see a window like this :

5.Name the App and select Platform and then click Add App.

6.Select ad format and name ad unit.

7.Set up Firebase Analytics (optional)

8.Click 'Done'

9.Repeat the process to add interstitial.

10.Go to Monetize tab and select the app you have created from the left side

11.Copy Ad Unit ID

12.Open 'Zombie Math.caproj' file and right click on the layout and select insert new object and add "AdmobAds". Once added you will see "AdmobAds" in Object types.

13.Paste Ad Unit ID for banner in 'Banner ID' box or Paste Ad Unit ID for interstitial 'Interstitial ID' box on the left side.

14.Save the Project.

15.Set 'Test Mode' to False If you want to export the for Android,IOS or Windows Phone.

If you want to export the game for Android,Go to this Link : https://www.scirra.com/tutorials/4848/how-to-get-admob-official-plugin-working-on-android-crosswalk/page-3

If you want to export the game for IOS, Watch this Tutorial : https://www.youtube.com/watch?v=dF903g4i9vo

If you want to export the game for Windows Phone,Go to this Link : https://www.scirra.com/tutorials/429/how-to-make-a-windows-phone-8-app
